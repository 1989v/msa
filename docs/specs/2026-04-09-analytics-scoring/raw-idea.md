# Raw Idea

검색/전시 영역에 CTR, CVR 등 지표 기반 컨텐츠 평가 체계 구현.

## 핵심 요구

1. **이벤트 수집**: 검색 키워드, 페이지 뷰, 상품 뷰, 주문 등 유형별 노출/클릭 로그 수집
2. **스코어 산출**: 이벤트 데이터 기반 지표별 스코어 산출 (정규화 포함), Kafka Streams 준실시간 처리
3. **검색 통합**: 키워드 검색 시 키워드 스코어 반영, 상품 정렬 시 상품 스코어 + 텍스트 스코어 조합
4. **개인화/추천 기반**: 지표 데이터를 토대로 향후 개인화/추천 확장 가능한 구조
5. **A/B 테스트 플랫폼**: 별도 공통 서비스, 실험 관리 + 버킷 할당 + 결과 분석 풀스택

## 사용자 결정

- 서비스 배치: analytics 신규 서비스
- 이벤트 저장소: Kafka + ClickHouse (OLAP)
- 스코어 산출 주기: 준실시간 (Kafka Streams)
- A/B 테스트: 풀스택 (실험 관리 + 버킷 + 결과 분석)
- Gateway에서 사용자 고유 식별 정보 발행
