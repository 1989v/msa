## 10. 결론

토스 수준은 **(a) UI 위계·인터랙션, (b) 데이터 풍부도, (c) 실시간성** 3축으로 갈린다.
- (a)는 1 사이클(2~3주)에 충분히 도달 가능
- (b)는 quant 백엔드 신규 ingest/API 다수 → 다단계 phase
- (c)는 SSE 인프라 + UI flash 효과 → 단일 phase

이 사이클 권장 스코프: **Phase 1 (Foundation: 차트엔진+헤더+토큰) + Phase 2 (정보탭+AI인사이트 카드) + Phase 3 (실시간 SSE)** 까지 단단히 잡고, 호가/매매주체/뉴스/발견화면은 후속 spec 으로 분리.
