## 사용 방식

- **주간 회독 일지**: `YYYY-WNN.md` 형식 (예: `2026-W18.md`)
- **회독 회차 단위 기록**: 어떤 주제 / 어느 파일 / 어디까지 / 발견 / 다음 액션
- **본 디렉토리는 git 관리** (학습 흔적은 보존). PII / 민감 정보 금지

---

## 템플릿

새 회독 주차 시작 시 [`template-week.md`](template-week.md) 복사:

```bash
cp study/log/template-week.md study/log/2026-WNN.md
```
