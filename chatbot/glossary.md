## 1. Bounded Context Overview

Chatbot BC는 **AI 챗봇 대화 관리 + Slack 연동**. Claude API를 AiModelPort로 추상화. Conversation Aggregate가 메시지 컬렉션을 보유하며 토큰 예산·접근 제어를 도메인 서비스로 처리. 내부(INTERNAL) 사용자와 외부(EXTERNAL) 사용자에 따라 정보 접근을 차별화한다.

## 2. Aggregates & Entities

### Conversation
**Type.** Aggregate
**Definition.** 사용자-챗봇 대화 흐름. channelType + externalChannelId + userId 결합으로 식별. Message 컬렉션 보유.
**Lifecycle.** `create(channelType, externalChannelId, userId, userRole)` → ACTIVE → {`close()` → CLOSED | `expire()` → EXPIRED}
**Invariants.**
- `externalChannelId.isNotBlank()`, `userId.isNotBlank()` (생성 시)
- `addMessage()` 는 ACTIVE 상태에서만 가능 — 종료된 대화에 메시지 추가 금지
- `close()` 는 ACTIVE 상태에서만 가능 — 중복 종료 차단
- `expire()` 는 ACTIVE 일 때만 전이 (이미 종료된 대화엔 무시) — **fail-silent**
- `lastActiveAt` 은 addMessage 호출 시 갱신
- `_messages` 는 private mutable — 외부에서 직접 변경 불가
**Related events.** — (현재 명시적 도메인 이벤트 없음)
**Avoid.** Session (일반 세션과 구분), Chat (의미 분화 시)
**Code.** `com.kgd.chatbot.domain.model.Conversation`
**Found in.** `chatbot/domain/model/`

### Message
**Type.** Entity (Conversation 내부)
**Definition.** Conversation 내 단일 발화. role + content + tokenCount + 선택적 cost.
**Lifecycle.** Conversation.addMessage 시 추가, 영구 (수정 메서드 없음)
**Invariants.**
- `content.isNotBlank()` (생성 시)
- USER 메시지는 `costUsd = null`
- ASSISTANT 메시지는 `costUsd` 필수
- 생성 후 모든 필드 불변
**Code.** `com.kgd.chatbot.domain.model.Message`

## 4. Domain Services

### ConversationDomainService
**Type.** Domain Service
**Definition.** Conversation 관련 stateless 도메인 로직.
**Code.** `com.kgd.chatbot.domain.service.ConversationDomainService`

**Methods.**

#### `buildContextWindow(conversation, maxTokens): List<Message>`
**Algorithm.** 최신 메시지부터 역순으로 누적, `totalTokens + message.tokenCount > maxTokens` 시 중단. AI 호출용 컨텍스트 윈도우 생성.

#### `validateAccess(userRole, query): AccessDecision`
**Algorithm.** INTERNAL 은 무조건 allow. EXTERNAL 은 `restrictedKeywords` 매치 시 deny.
**Restricted keywords.** `private, internal, secret, credential, env, password, token, api key, 비밀, 인증키` (대소문자 무시)

#### `estimateTokenCount(text): Int`
**Algorithm.** `text.length / 3.5` (한국어+영어 혼합 추정, 최소 1).
